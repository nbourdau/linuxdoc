linuxdoc.autodoc module¶

autodoc¶

Implementation of the kernel-autodoc command.

copyright:Copyright (C) 2016 Markus Heiser
license:GPL Version 2, June 1991 see Linux/COPYING for details.

The kernel-autodoc command extracts documentation from Linux kernel’s source code comments, see --help:

$ kernel-autodoc --help

Note

The kernel-autodoc command is under construction, no stable release yet. The command line arguments might be changed/extended in the near future.
